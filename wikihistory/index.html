    维基编年史¶

    声明¶

    以下只是笔者对于海弗拉斯维基发展历史的推测，若有更全面的线索或者更权威的说法请在 GitHub 联系笔者。

    海弗拉斯维基 (2009-2011)¶

    根据百度百科上的说法，海弗拉斯维基是属于杭外人自己的维基，希望提供关于杭外的各种官方的，有趣的，有益的信息，注重信息的交互与更新，兼顾趣味性与实用性，由一位即将毕业的名叫Omi的学生在2009年4月25日创建，为了在毕业后更好地记住杭外，并帮助更多的杭外人。通过 Wayback Machine，笔者得知，从2010年9月24日到2011年2月8日，在 hflswiki.com 的域名上，的确出现了一个名为海弗拉斯维基 (HFLSWiki) 的网站。 (但是根据百度贴吧上很多人的表现和说法，海弗拉斯维基上一次出现还是在2009年，2010年到2011年的海弗拉斯维基的历史不存在；奇怪的是，在百度贴吧上又有人转载2010年首发的海弗拉斯维基的帖子。)

    词条内容多轻松为主，基本都是人物小传和对于同学老师的调侃；通过 bbs.hflswiki.com 这个域名的使用，我们似乎可以推测，海弗拉斯维基有着自己的论坛。

    我们能够推出以下结论： 1. “海弗拉斯”这个名称的使用绝对可以推到 2009 年以前。 2. 这个网站和任何官方的组织或社团毫无关系，其本质是个人网站。 3. 至少在 2009 年到 2011 年这段时间内，这个版本的海弗拉斯维基存在且运行良好。

    杭外维基百科 (2013-2014)¶

    根据百度百科上的说法，在 2013 年，杭外 Binary电子社购买了 hflswiki.com 的域名，被遗弃多年 (似乎也就两年左右，贴吧上也有说法是将近四年) 的海弗拉斯维基，如今改名为杭外维基百科。在海弗拉斯吧，他们还宣称要在 2013 年 5 月举办“杭外维基编辑大赛”，结果不详。据百度百科，在网站注册过的杭外人超 200 人，日浏览量超 2000，定位是“杭外社交百科网站”。根据 Binary电子社内部人员统计，截至 2013 年 6 月，活跃会员（一周上线 2 次以上）已达230人，精品词条逾 100 条。2014 年 5 月之后，通过 Wayback Machine 确定域名到期，网站倒闭。

    词条内容也是多轻松为主，也还基本都是人物小传和对于同学老师的调侃；除此之外，此维基还有许多美食条目，以下是被笔者抢救出来的一条。

    HFLS.WIKI (2019 - Now)¶

    根据网站首页的描述，这是一个由HFLS的学生 (即 DoraDmon) “自愿搭建”的小型校园Wiki。我们并不知道它与前两者有无联系，根据 GitHub 上的仓库，可以合理推测，这是一个个人静态网站，采用了 MKDocs 技术构建。由于在 2019 年的活跃，GitHub 将其收录到了 2020 年的南极代码库中。根据 GitHub 上 Issue #1 的说法，网站的所有权和域名已经转手，与原作者无关，而内容从 2019 年后便已没有更新。

    杭外剑高首次被包括在各种形式的维基中，被称为“剑桥A级中心”或“CAL中心”。此外，虽然内容不多，但此维基较为客观准确，整合了大量的真实内容。

    海弗拉斯 Wiki (2024 - Now)¶

    笔者认为这是一个属于杭州外国语学校的独立校园 Wiki，但它将不仅仅是一个独立的 Wiki。如它的先祖一样，它将有一个完整的生态，截至 2024 年 8 月 28 日，HFLS.top 系网站已达到 3 个。笔者期待它将取得比前辈们更加非凡的成就。
